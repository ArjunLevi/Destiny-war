# Destiny War

Base mini app deployed at [destinywar.app](https://destinywar.app).

## Run locally

```bash
npm install
copy .env.local.example .env.local   # Windows
npm run dev
```

Set `NEXT_PUBLIC_URL=https://destinywar.app` in Vercel env vars for production.

See [docs/SETUP.md](docs/SETUP.md) for contracts, Farcaster manifest, and deployment steps.
